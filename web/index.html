LinkC 轻量级的在线同步聊天软件

开放源码，统一协议以及编程API
项目持续开发中

  • 基本信息

LinkC 是一个有着完全独立的P2P协议，拥有简单的即时通讯和登陆登出功能在内的轻量级软件，LinkC提供GNU/Linux，MAC OSX，Windows下的客户端。采用Qt作为客户端的图形库，力求在不同平台上给用户统一的体验

  • 开发进度

现目前 LinkC 完成度不高。包括好友搜索，账户注册在内的一些功能还没有添加。对于不同NAT设备的通讯方式还未完备

  • 审阅代码

LinkC 属于开源软件，欢迎任何人来审阅 LinkC 的代码。现目前，LinkC 的代码被托管在 Github 上
